# Codex Workspace Guide

This folder holds the persistent agent scaffold for `microGo`.

## Layout

- `agents/`: specialist instruction files
- `memories/`: short-lived project memory that should stay current
- `workflows/`: orchestration rules and reusable prompt templates
- `PROMPT_GUIDE.md`: most-used prompts for driving the system
